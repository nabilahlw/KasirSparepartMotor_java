## ⚙️ Cara Menjalankan

### Prasyarat

Pastikan **Java JDK** sudah terinstall. Cek dengan:
```bash
java -version
```
Jika belum ada → download di [oracle.com/java](https://www.oracle.com/java/technologies/downloads/)

### Opsi A: VS Code (Rekomendasi)

**Step 1** — Install extension **"Extension Pack for Java"** di VS Code

**Step 2** — Clone atau download repo ini, lalu buka foldernya di VS Code

**Step 3** — Buka file `KasirSparepart.java`

**Step 4** — Di terminal VS Code, jalankan:
```bash
javac KasirSparepart.java
java kasirSparepart.KasirSparepart
```

### Opsi B: Terminal / CMD / PowerShell

**Step 1** — Clone repo:
```bash
git clone https://github.com/username/KasirSparepartMotor_java.git
```

**Step 2** — Masuk ke folder yang berisi file `.java`:
```bash
cd KasirSparepartMotor_java
```

**Step 3** — Compile:
```bash
javac KasirSparepart.java
```

**Step 4** — Jalankan:
```bash
java kasirSparepart.KasirSparepart
```
